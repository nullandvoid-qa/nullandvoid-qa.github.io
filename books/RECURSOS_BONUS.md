## 🛠️ Ferramentas Essenciais

- **Streaks:** Simples, focado em correntes (iOS)
- **Todoist:** Tem seção de hábitos robusta
- **Notion:** Flexível, pode criar templates personalizados
- **Obsidian:** Para journaling de identidade

### **Time Blocking (Bloqueio de Tempo)**
- **Google Calendar:** Bloqueio visual de tempo essencial
- **Outlook Calendar:** Similar ao Google, para usuários Microsoft
- **Calendly:** Para agendar tempo de foco
- **Fantastical:** Calendar avançado com blocking

### **Environment Design (Design Ambiental)**
- **Trello:** Para planear layout do ambiente
- **Figma:** Design visual de espaços
- **Pinterest:** Inspiração de design ambiental
- **RoomSketcher:** Layout de espaços físicos

### **Note-taking & Journaling**
- **Notion:** All-in-one para templates, journaling, tracking
- **Obsidian:** Zettelkasten para conexões entre conceitos
- **Roam Research:** Similar ao Obsidian, para conexões
- **Evernote:** Clássico, organização robusta
- **Bear:** Minimalista, para usuários Apple

### **Reading & Learning**
- **Kindle:** Leitura digital com highlights

- **Readwise:** Sincroniza highlights de Kindle, web, etc.
- **Medium:** Artigos complementares aos livros

## 📋 Templates Prontos

### **Template de Identidade Journal**
```markdown
## Identidade Journal - [Data]

### Quem sou eu hoje?
- [ ] Sou alguém que [hábito 1]
- [ ] Sou alguém que [hábito 2]
- [ ] Sou alguém que [hábito 3]

### Votos de Identidade Hoje
Ação 1: [ ] Voto para "sou alguém que faz X"
Ação 2: [ ] Voto para "sou alguém que faz Y"
Ação 3: [ ] Voto para "sou alguém que faz Z"

### Reflexão
Hoje eu me senti [conectado/desconectado] da minha identidade alvo porque:
```

### **Template de Critério de 90% (Essencialismo)**
```markdown
## Critério de 90% - [Oportunidade]

1. Isso alinha com meus 3 objetivos principais? [ ] Sim [ ] Não
2. Isso usa minhas forças únicas? [ ] Sim [ ] Não
3. Isso gera impacto duradouro ou apenas temporário? [ ] Duradouro [ ] Temporário
4. Isso pode ser feito de forma excepcional? [ ] Sim [ ] Não
5. Meu entusiasmo é "Hell Yeah"? [ ] Sim [ ] Não

**Score:** ___/5
**Decisão:** [ ] SIM (4-5/5) [ ] NÃO (0-3/5)
```

### **Template de Habit Stacking**
```markdown
## Habit Stack - [Nome]

### Gatilho Existentes
1. [Hábito existente 1]
2. [Hábito existente 2]
3. [Hábito existente 3]

### Novos Hábitos a Adicionar
- Depois de [Gatilho 1], vou [Novo Hábito 1]
- Depois de [Gatilho 2], vou [Novo Hábito 2]
- Depois de [Gatilho 3], vou [Novo Hábito 3]

### Fricção Eliminada
- [ ] Gatilho óbvio (visualmente)
- [ ] Hábito fácil (<2 minutos)
- [ ] Recompensa imediata definida
```

### **Template de Flow State Design**
```markdown
## Flow State Design - [Atividade]

### Pré-requisitos
- [ ] Objetivo claro: [ ]
- [ ] Feedback imediato: [ ]
- [ ] Desafio apropriado: [ ]

### Ambiente
- [ ] Sem interrupções (telefone, notificações)
- [ ] Tempo bloqueado: [ ]
- [ ] Ferramentas prontas: [ ]

### Execução
- [ ] Começar com 5 min "easy start"
- [ ] Aumentar gradualmente o desafio
- [ ] Documentar quando entra em flow
```

## 🎯 Exercícios Práticos Adicionais

### **Exercício 1: Identidade Audit (1 hora)**
**Objetivo:** Entender sua identidade atual vs alvo

**Passos:**
1. Liste 10 identidades que você tem hoje (ex: "sou procrastinador", "sou organizado")
2. Liste 10 identidades que você quer ter (ex: "sou produtivo", "sou saudável")
3. Identifique gaps entre atual e alvo
4. Para cada gap, escolha 1 hábito para construir

**Resultado:** Clareza sobre quem você é vs quem quer ser.

### **Exercício 2: Environment Audit (2 horas)**
**Objetivo:** Identificar onde seu ambiente sabotage ou apoia hábitos

**Passos:**
1. Escolha um ambiente (casa, escritório, digital)
2. Identifique 5 coisas que sabotam hábitos desejados
3. Identifique 5 coisas que apoiam hábitos desejados
4. Plano: eliminar 2 sabotadores, fortalecer 2 apoios

**Resultado:** Ambiente desenhado para sucesso.

### **Exercício 3: Time Audit (1 semana)**
**Objetivo:** Entender onde seu tempo realmente vai

**Passos:**
1. Por 7 dias, documente cada 30 minutos do seu dia
2. Categorize cada bloco: essencial / delegável / eliminável
3. Calcule % de tempo em cada categoria
4. Plano: eliminar 20% de não-essenciais

**Resultado:** Clarity brutal sobre uso de tempo.

### **Exercício 4: Financial Audit (1 dia)**
**Objetivo:** Entender sua situação financeira

**Passos:**
1. Liste todos os ativos (coisas que colocam dinheiro no bolso)
2. Liste todos os passivos (coisas que tiram dinheiro do bolso)
3. Calcule net worth (ativos - passivos)
4. Identifique 1 ativo para construir, 1 passivo para eliminar

**Resultado:** Baseline financeiro claro.
