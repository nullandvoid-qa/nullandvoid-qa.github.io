<h2>iOS Emulator para Testes</h2>

<h3>Objetivos de Aprendizado</h3>
<p>Após esta aula, você será capaz de:</p>
<ul>
  <li>Configurar e usar o iOS Simulator no macOS</li>
  <li>Entender as diferenças entre simulator e device real</li>
  <li>Executar testes automatizados no iOS Simulator</li>
  <li>Diagnosticar problemas comuns no emulador iOS</li>
</ul>

<h3>Resumo Executivo</h3>
<p>O iOS Simulator é a ferramenta oficial da Apple para testar apps iOS sem precisar de dispositivos físicos. Ele roda apenas no macOS e é essencial para desenvolvimento e testes de apps iOS. Embora não substitua completamente testes em dispositivos reais, o simulator cobre ~80% dos cenários de teste com muito mais eficiência.</p>

<h3>Configuração Inicial</h3>

<h4>Requisitos</h4>
<ul style="margin:1rem 0">
  <li><strong>Mac com chip Intel ou Apple Silicon</strong> (M1/M2/M3)</li>
  <li><strong>Xcode</strong> (disponível na Mac App Store, gratuito)</li>
  <li><strong>Command Line Tools for Xcode</strong></li>
</ul>

<h4>Instalação</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# 1. Instale Xcode da Mac App Store
# 2. Instale Command Line Tools
xcode-select --install

# 3. Aceite a licença Xcode (primeira vez)
sudo xcodebuild -license accept

# 4. Verifique simulators disponíveis
xcrun simctl list devices available
</pre>

<h3>Usando o iOS Simulator</h3>

<h4>Linha de Comando (simctl)</h4>
<p>O <code>simctl</code> é a ferramenta de linha de comando para controlar simulators:</p>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Listar todos os devices disponíveis
xcrun simctl list devices

# Listar apenas devices disponíveis
xcrun simctl list devices available

# Bootar um simulator específico
xcrun simctl boot "iPhone 15 Pro"

# Abrir o app Simulator
open -a Simulator

# Instalar um app no simulator
xcrun simctl install booted /path/to/app.app

# Launchar um app
xcrun simctl launch booted com.bundle.identifier

# Desligar simulator
xcrun simctl shutdown booted

# Resetar simulator para estado de fábrica
xcrun simctl erase all
</pre>

<h4>Gerenciamento de Devices</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Criar um novo simulator
xcrun simctl create "My iPhone 15" "iPhone 15" "iOS 17.0"

# Deletar um simulator
xcrun simctl delete &lt;device_udid&gt;

# Renomear simulator
xcrun simctl rename &lt;device_udid&gt; "New Name"

# Adicionar fotos ao simulator
xcrun simctl addphoto booted /path/to/photo.jpg

# Adicionar contatos
xcrun simctl addcontact booted /path/to/contact.vcard
</pre>

<h3>Testes Automatizados no iOS Simulator</h3>

<h4>Com XCUITest (Framework Oficial)</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Rodar testes UI no simulator
xcodebuild test \
  -project MyProject.xcodeproj \
  -scheme MyScheme \
  -destination 'platform=iOS Simulator,name=iPhone 15 Pro'

# Rodar testes em múltiplos simulators
xcodebuild test \
  -project MyProject.xcodeproj \
  -scheme MyScheme \
  -destination 'platform=iOS Simulator,name=iPhone 15' \
  -destination 'platform=iOS Simulator,name=iPhone SE'

# Rodar testes específicos
xcodebuild test \
  -project MyProject.xcodeproj \
  -scheme MyScheme \
  -destination 'platform=iOS Simulator,name=iPhone 15 Pro' \
  -only-testing:MyTestSuite/testMySpecificTest
</pre>

<h4>Com Appium (Cross-Platform)</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# capabilities para iOS Simulator
{
  "platformName": "iOS",
  "platformVersion": "17.0",
  "deviceName": "iPhone 15 Pro",
  "automationName": "XCUITest",
  "app": "/path/to/app.app",
  "noReset": true
}

# Comando para iniciar Appium
appium --address 127.0.0.1 --port 4723

# Exemplo de teste em Python
from appium import webdriver

caps = {
    "platformName": "iOS",
    "platformVersion": "17.0",
    "deviceName": "iPhone 15 Pro",
    "automationName": "XCUITest",
    "app": "/path/to/app.app"
}

driver = webdriver.Remote("http://127.0.0.1:4723/wd/hub", caps)
</pre>

<h3>Configurações Avançadas</h3>

<h4>Simular Condições de Rede</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Simular diferentes condições de rede
xcrun simctl status_bar booted setCellularDataMode --4g
xcrun simctl status_bar booted setCellularDataMode --3g
xcrun simctl status_bar booted setCellularDataMode --edge
xcrun simctl status_bar booted setCellularDataMode --wifi

# Simular airplane mode
xcrun simctl status_bar booted setCellularBars --no-service
</pre>

<h4>Simular Localização</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Definir localização específica
xcrun simctl location booted set 37.7749,-122.4194  # San Francisco

# Ativar/desativar GPS
xcrun simctl location booted enable
xcrun simctl location booted disable
</pre>

<h4>Simular Notificações</h4>
<pre style="background:#f5f5f5; padding:1rem; border-radius:0.5rem; overflow-x:auto">
# Enviar notificação push
xcrun simctl push booted com.bundle.identifier /path/to/payload.json

# Exemplo de payload.json
{
  "aps": {
    "alert": "Test notification",
    "badge": 1,
    "sound": "default"
  }
}
</pre>

<h3>Troubleshooting Comum</h3>

<div style="background:#fef3c7; padding:1.5rem; border-radius:0.5rem; margin:1.5rem 0; border-left:4px solid #f59e0b">
<p><strong>Problema 1: Simulator não boota</strong></p>
<ul style="margin:1rem 0">
  <li>Verifique se o Xcode está instalado corretamente</li>
  <li>Execute <code>xcode-select --print-path</code> para verificar o path</li>
  <li>Reinicie o simulator: <code>xcrun simctl shutdown all</code></li>
</ul>

<p><strong>Problema 2: App não instala</strong></p>
<ul style="margin:1rem 0">
  <li>Verifique se o app é assinado corretamente</li>
  <li>Certifique-se de que o iOS version é compatível</li>
  <li>Use <code>codesign -dv /path/to/app.app</code> para verificar assinatura</li>
</ul>

<p><strong>Problema 3: Testes falham no simulator mas funcionam no device</strong></p>
<ul style="margin:1rem 0">
  <li>Simulator não tem hardware real (GPS, acelerômetro, etc)</li>
  <li>Performance do simulator é diferente do device real</li>
  <li>Alguns APIs de sistema se comportam diferente no simulator</li>
</ul>
</div>

<h3>Melhores Práticas</h3>
<ul style="margin:1rem 0">
  <li><strong>Use simulators para testes de regressão</strong> - são rápidos e gratuitos</li>
  <li><strong>Teste em múltiplos iOS versions</strong> - iOS 15, 16, 17</li>
  <li><strong>Combine com devices reais</strong> - para performance e hardware-specific features</li>
  <li><strong>Automate setup</strong> - scripts para bootar e configurar simulators</li>
  <li><strong>Limpe simulators regularmente</strong> - <code>xcrun simctl erase all</code></li>
</ul>
