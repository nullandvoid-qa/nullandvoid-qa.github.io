# Formulário de Feedback — Beta

Objetivo: coletar feedback qualitativo e quantitativo de 5 usuários para validar usabilidade, clareza das aulas e exercícios.

Mensagem de convite (use no e-mail/discord):

Olá — estamos testando uma versão beta das aulas de QA. Você pode nos ajudar com um rápido feedback (5–10 minutos)? Em troca, oferecemos acesso antecipado e um certificado de participação.

Link para o formulário: (preencher com URL)

Perguntas sugeridas:

1. Qual sua experiência geral com as aulas testadas? (1-5)
2. As aulas foram claras e fáceis de seguir? (1-5)
3. Os objetivos de aprendizado foram alcançados? (Sim/Parcial/Não)
4. Ao menos um exercício prático foi útil? (Sim/Não). Se não, explique.
5. O material de apoio (links, exemplos) foi suficiente? (Sim/Não)
6. Você encontrou erros, links quebrados ou instruções confusas? (campo de texto)
7. Sugestões para melhorar a aula (campo de texto)
8. Você aceita ser contatado para uma entrevista rápida? (Sim/Não) — Se sim, deixe contato.

Instruções para o time: coletem as respostas em `reports/beta-feedback.json` e resumam pontos comuns antes de priorizar correções.
