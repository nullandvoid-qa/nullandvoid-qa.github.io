# Exercício: Papéis: QA, QC e Tester (l2)

Objetivo: aplicar conceitos de papéis em cenários práticos.

Exercício:

1. Classifique as seguintes atividades como QA, QC ou Tester:
   - Escrever um teste de integração automatizado.
   - Participar do refinement apontando ambiguidades nos requisitos.
   - Executar um caso de teste manual e registrar evidências.

Gabarito exemplo:

- Escrever um teste de integração automatizado — Tester
- Participar do refinement — QA
- Executar caso de teste manual — QC/Tester (reportar como QC)
