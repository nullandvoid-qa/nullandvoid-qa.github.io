# Exercício: Critérios de Aceitação e Definição de Pronto (l9)

Objetivo: escrever critérios de aceitação claros e itens de Definition of Done.

Exercício:

1. Para a história "Como usuário, quero cadastrar meu cartão para pagar compras", escreva 3 critérios de aceitação e 3 itens de Definição de Pronto.

Gabarito exemplo:

- Critérios de aceitação:
  - O usuário vê mensagem de confirmação ao salvar o cartão.
  - Número do cartão validado por algoritmo Luhn.
  - Data de validade aceita apenas em formato MM/AA.

- Definição de Pronto:
  - Código revisado e aprovado.
  - Testes automatizados adicionados para fluxo de pagamento.
  - Documentação de uso atualizada.
