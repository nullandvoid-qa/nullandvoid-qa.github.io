# Exercício: SDLC e onde o QA se encaixa (l3)

Objetivo: mapear atividades de QA em fases do SDLC.

Exercício:

1. Para um app de e-commerce, identifique em qual fase do SDLC cada uma das ações abaixo deveria ocorrer e descreva uma atividade de QA correspondente:
   - Definição de requisitos para checkout
   - Implementação da API de pagamentos
   - Deploy para produção

Gabarito exemplo:

- Definição de requisitos — atividade QA: definir critérios de aceitação e casos de teste no refinement.
- Implementação da API — atividade QA: revisão de contrato, testes de integração e automação de regressão.
- Deploy — atividade QA: smoke tests e monitoramento de produção (health checks).
