# Exercício: Testes funcionais vs não-funcionais (l4)

Objetivo: praticar a escolha de tipos de teste para uma feature.

Exercício:

1. Para a feature "checkout", escreva:
   - 2 testes funcionais
   - 1 teste de performance (descrição simples)
   - 1 teste de segurança (descrição simples)

Gabarito exemplo:

- Testes funcionais:
  - Validar que o usuário consegue adicionar item ao carrinho e finalizar compra.
  - Validar que descontos são aplicados corretamente.
- Teste de performance:
  - Medir tempo de carregamento da página de checkout sob carga de 100 usuários simultâneos (objetivo < 2s).
- Teste de segurança:
  - Verificar que dados do cartão não são expostos no DOM e são transmitidos por HTTPS.
