# Checklist de Qualidade das Aulas

Este checklist serve como critério mínimo para que uma aula seja considerada competitiva frente a cursos pagos.

1. Conteúdo
   - Título claro e objetivos de aprendizado listados
   - Estrutura lógica (introdução, teoria, exemplos, resumo)
   - Fontes e referências atualizadas
   - Duração estimada indicada

2. Exercícios e Avaliações
   - Pelo menos 1 exercício prático por aula
   - Gabaritos ou instruções para correção disponíveis
   - Ao menos um desafio de nivelamento (fácil/médio/difícil)

3. Material de Apoio
   - Links e leituras adicionais
   - Arquivos de exemplo ou repositório quando aplicável
   - Transcrições ou notas de aula

4. Acessibilidade e Usabilidade
   - Texto legível e sem jargões desnecessários
   - Imagens com `alt` e legendas quando relevantes
   - Cabeçalhos e estrutura semântica (HTML) apropriados

5. Avaliação de Qualidade (meta)
   - Aula revisada por pelo menos 1 revisor técnico
   - Evidência de execução (screenshots, gravações ou reproduções)
   - Métricas de sucesso sugeridas (tempo médio, taxa de conclusão)

6. Publicação e Manutenção
   - Data da última revisão registrada
   - Checklist de release (links funcionando, imagens carregam)
   - Responsável pela manutenção identificado

7. Critérios de Aprovação
   - Cumpre 80% dos itens acima para publicação inicial
   - Cumpre 100% após revisão e feedback de usuários beta

Use este documento como referência na tarefa `Definir checklist de qualidade`.
